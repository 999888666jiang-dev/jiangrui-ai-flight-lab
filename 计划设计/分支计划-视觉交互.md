# 分支计划 - 视觉交互

## 目标

网站要像高级互动网页一样有记忆点和空间感，但不能直接复制参考网站或参考视频。设计语言必须服务于姜睿的能力身份：AI 设计工程、FPV、无人机全机型操控、程序开发、无人机测试。

## 参考来源

1. Active Theory
   - 提炼点：暗色空间、WebGL 主视觉、极简导航、少量强信息、沉浸式画面。
   - 不复制：具体布局、作品内容、图形资产和品牌语言。

2. 用户提供参考视频
   - 提炼点：全屏主体、材质生长/变形、鼠标与滚动驱动、文字浮在前景。
   - 不复制：树枝、苔藓、字体、原文案、原站点构图。

## 当前视觉方案

概念名称：AI Flight Lab

核心元素：

- 透明科技感桨叶
- 青绿色航迹粒子
- 中区飞控姿态解算粒子
- HUD 刻度和飞控数据
- 暗色实验室背景
- 开放式信息带和证据节点

当前页面结构：

1. 首屏：姜睿 / AI Flight Lab / 身份定位 / 透明桨叶 / FPV 照片。
2. 任务：说明页面不是模板简历，而是能力证据实验室。
3. 系统：AI 设计、FPV、全机型无人机、代码能力组成能力矩阵。
4. 航迹：真实经历时间线。
5. Video Bay：独立视频舱子页面，预留真实 FPV/无人机视频背景。
6. Evidence Vault：独立证据库子页面，集中展示执照、二维码、简历 PDF 和后续资源入口。
7. 六大展示网页：从 Evidence Vault 进入 FPV 飞行视频、AI 设计作品、全机型无人机图库、代码与项目、飞行日志与测试报告、证书与荣誉六个独立展示页。
8. 联系：微信二维码和简历 PDF。

## 后续可升级方向

1. 证据库真实素材点亮
   - 当前六个展示页已建立入口、路由、占位和独立动效。
   - 后续资源到位后，按 slug 接入封面、视频、图片、PDF、项目链接和说明。
   - 素材上线时可做“节点上线 / 资源校验完成”动画，不改变六个入口结构。

2. 视频进入方式
   - 当前 Video Bay 不请求缺失视频。
   - 后续 FPV 视频可做静音背景循环、封面图和移动端轻量替代。

3. 飞行日志可视化
   - 可将测试记录做成航迹曲线、任务卡、异常点标注。

4. 更精细的桨叶材质
   - 当前是代码生成透明桨叶。
   - 后续可加入更细的边缘高光、内部线路、速度残影。

5. 侧边粒子接入真实数据
   - 当前侧边粒子根据滚动和路由 preset 响应。
   - 后续可接入真实飞行日志、任务编号或视频段落状态。

6. 中区粒子接入真实飞行状态
   - 当前中区粒子根据鼠标、滚动和路由 preset 响应。
   - 后续可接入真实飞控参数、任务坐标、测试数据或视频章节状态。

7. 展示页局部动效深化
   - FPV 页可接入真实飞行视频后强化速度线、姿态地平线和 REC 状态。
   - AI 页可接入作品图后让神经节点与作品流关联。
   - 机库页可接入照片墙后让机型分类和图片灯扫联动。
   - 成交成果页可继续接入图片、说明和阶段标签，让展柜从纯视频升级为成果案例库。
   - 日志页可接入报告后把 PDF、表格和问题复盘做成黑匣子时间线。
   - 证书页可接入更多荣誉后做核验时间线和扫描通过状态。

## 设计约束

- 不使用紫色、米色、橙色作为主风格。
- 不使用装饰性光球、模板卡片堆叠、过度霓虹。
- 不卡住移动端首屏。
- 不为了炫技牺牲内容可读性。
- 所有动效必须有降级路径。
- 2026-06-27 起，所有新增动效必须读取统一 `AdaptationProfile`：桌面完整、平板降级、手机轻量、微信极轻、`prefers-reduced-motion` 静态或近静态。
- 新增 fixed HUD、灯箱、媒体控制、播放遮罩和全屏视觉层必须使用 safe-area 变量，不能再单独硬编码刘海屏/底部安全区偏移。
## 2026-06-27 证书书本展厅与药丸导航补充

- 导航由普通文字链接升级为药丸导航：active 胶囊、hover 液态扩散、双层文字滑动和移动端弹层收纳。
- 该效果按 Vue 原生实现，不引入 React Bits 原组件，不新增 OGL。
- `证书与荣誉` 页面视觉从重 HUD 验证舱调整为轻量书本展厅：纸张底板、书脊、压印、细光边和书页打开态。
- 证书卡片 hover 只做轻微开合和抬升，避免过重的 AI 模块感。
- 打开态保留键盘可用性和 reduced-motion 降级；低动态模式下关闭强翻页动画。
- 证据库入口中的 `证书与荣誉` 卡片接入真实证书封面，视觉上从空占位变成可感知的成果入口。
## 2026-06-27 全机型无人机图库 Hangar Dome 补充

- `全机型无人机图库` 详情页采用“数字机库穹顶”视觉：桌面端图片围绕中心形成可拖拽、可惯性旋转、可聚焦吸附的 CSS 3D dome。
- 交互参考 DomeGallery 的“环绕、拖拽、惯性、点击放大”思路，但实现为 Vue 原生组件，不引入 React、`@use-gesture/react` 或额外 3D 依赖。
- 页面动效层级保持克制：主穹顶负责炫技和浏览，底部网格负责高效查找，灯箱负责高清查看，避免所有图片卡片都做重动效。
- 移动端、微信、触摸端和低性能设备自动降级为横向精选轨道 + 瀑布流，不启用 3D 穹顶和惯性，保证稳定性优先。
- `prefers-reduced-motion` 下禁用强过渡和惯性，只保留清晰图片、轻边框和基础点击灯箱。
- 证据库入口卡片使用真实图库 WebP 封面替代空网格占位，仍保留原吸入式进入动画。

## 2026-06-28 首页挂绳证件卡补充

- 首页头像区由静态 HUD 相框升级为“Flight Pass 挂绳证件卡”，视觉上更接近真实工牌、飞行实验室通行证和现场证据卡。
- 组件使用 CSS 3D + SVG 绳带 + GSAP CSS 变量完成拖拽、回弹、hover 倾斜、扫光和露背信息态，不增加新的第三方依赖。
- 桌面宽屏允许鼠标/触控笔拖拽，真实 touch pointer 不触发自由拖拽，避免手机滚动冲突。
- 持续动效只作用于内部证件面与挂绳细节，外层按钮保持稳定，避免交互目标在自动化和真实点击中漂移。
- 证件卡位置已避开 ALT/SPD/HDG telemetry，桌面端不遮挡读数；移动端改为正常文档流，不产生横向溢出。
- reduced-motion 下关闭摆动、扫光和拖拽回弹，只保留静态证件质感。
- 追加入场动效：页面打开后证件卡先从视口上方落入舱位，落点有短暂过冲和回弹，像挂绳卡被放下后自然晃定。
- 拖拽重心改为纵向弹性：上下位移幅度更明显，横向只保留轻微摆动，降低对 Hero 主画面的干扰。
- 绳带张力随拖拽方向变化：向下拉伸、向上压缩，释放后先轻微反向回弹再归位。

## 2026-06-28 首页单链条挂件补充

- 双股挂绳视觉偏厚、偏突兀，已替换为“棚顶单链条 + 金属挂钩 + 证件卡”的轻量挂件结构。
- 链条由细小金属链节组成，顶部增加棚轨和螺丝点，材质以石墨黑、金属灰、低饱和青绿边光为主，减少硬 HUD 和 AI 模块感。
- 桌面拖拽从受限纵向拉伸升级为可移开式自由拖动：横向最大约 280px、纵向最大约 240px，用户可以临时把证件卡拖离后方内容区域。
- 链条和挂钩会跟随拖拽方向滑移、倾斜和拉伸，避免卡片大幅移动时与挂点断开；松手后通过 GSAP 弹性回摆归位。
- 触摸端、微信、低性能和 reduced-motion 下不启用自由拖动，只保留静态质感和基础点击态，保证移动端滚动优先。

## 2026-06-28 首页黑色织带与五寸桨叶补充

- 最终视觉继续向参考 Lanyard 靠拢：不使用金属链节，改为黑色织带从画面上方垂入，带细缝、暗纹和白色小桨叶标记。
- 挂点结构由“棚轨 + 链条”改为“织带 + 金属圆环 + 短连接绳 + 卡片固定孔”，物件关系更清晰，也更像真实 FPV 通行证。
- 卡片背后加入三叶五寸桨叶作为局部主视觉：半透明青绿色材质、虚线电机环、叶片内骨线和慢速旋转，强化无人机能力标签。
- 五寸桨叶必须位于卡片后层，只从卡片边缘露出，不遮挡 FPV 照片主体、底部身份条和 Hero 主文案。
- 桌面端保留大幅拖拽与回弹；织带和固定点跟随卡片倾斜，确保拖动时仍像一个被挂住的真实物件。
- 移动端和 reduced-motion 下关闭持续旋转与自由拖动，只保留静态织带和桨叶层，避免滚动冲突。

## 2026-06-28 字体气质与资料库命名补充

- 首页能力、首页经历、资料库首页三处大标题不再沿用通用超大字号，而是使用专用排版类控制 `max-width`、`line-height`、`word-break` 和移动端字号。
- 中文标题采用更短、更有节奏的表达，减少一行内字符密度；英文同步改为自然作品集语境。
- 资料库命名从“证据库 / Evidence Vault”切换为“资料库 / Resource Library”，视觉语气更像个人素材与能力档案中心，减少重安全舱和重 AI 模块感。
- 媒体空态从 `PENDING` 调整为 `STANDBY`，语义保持“资源未接入时安全待命”，但减少临时模板感。
- 本轮只做局部覆盖，不修改 Hero、视频、图库、证书、挂绳、桨叶和粒子主视觉结构。
