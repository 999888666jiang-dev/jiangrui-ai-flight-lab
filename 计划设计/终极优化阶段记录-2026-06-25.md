# 终极优化阶段记录 - 2026-06-25

本阶段根据用户要求，将姜睿个人网站从“高级简历页面”继续推进为“AI Flight Lab / Mission Control”交互体验。设计参考来自前一阶段 7 个网站研究成果，但本次实现没有复制任何参考站的品牌、文案、素材或专属布局，只迁移其设计思路：沉浸式首屏、任务侧轨、字体冲击、可点击系统、视频舞台和证据库。

## 已完成实现

- 新增全局任务侧轨 `MissionRail`，显示 TOP / BRIEF / SYSTEMS / MISSIONS / VIDEO / VAULT / CONTACT，并用滚动进度条表现浏览进度。
- 首屏继续保留 Three.js 透明科技感桨叶，并升级为四叶科技桨叶与 HUD 风格首屏。
- 首屏新增 Mission Ready 状态条、AI / FPV / UAV / CODE 身份组和系统化视觉层。
- 能力系统从静态卡片升级为可点击能力矩阵，点击 AI / FPV / UAV / 程序节点会切换当前系统读数。
- 航迹时间线前新增 Mission Control 任务控制台，点击不同经历会切换右侧大窗口详情。
- 新增 `VideoBaySection`，作为可接入视频背景的子页面/全屏章节。
- 证据库与联系区保留真实资源优先原则，不伪造项目、视频或证书。
- 新增 hash 直达滚动兜底，`#video-bay` 等锚点在桌面和移动端都能落到目标区。
- 已把生成的视觉概念图保存到 `design-concepts/ai-flight-lab-terminal-concept-2026-06-25.png`。

## 视频素材要求

视频背景预留路径：`public/videos/fpv-lab-background.mp4`

建议视频风格：

- 8-15 秒无缝循环。
- FPV 或无人机主观视角。
- 夜航、城市、山谷、起降、穿越、低空巡航均可。
- 画面稳定、有速度感、少字幕、无水印。
- 中心区域不要有重要主体，方便叠加 HUD 与文字。
- 色调建议偏冷、黑场、青绿/白光点缀，避免过强暖色和过多花哨字幕。

当前未主动请求缺失视频文件，避免页面产生 404 网络噪声。用户补充视频后再接入真实背景视频。
