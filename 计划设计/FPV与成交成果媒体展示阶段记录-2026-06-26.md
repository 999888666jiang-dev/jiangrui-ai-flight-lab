## 风险与处理

| 风险 | 处理 |
| --- | --- |
| 视频总体积较大 | 只在详情页加载当前主视频，集合页点击卡片后才加载对应视频 |
| 中文文件名导致浏览器请求失败 | 公开文件名统一改为 ASCII，manifest 保留原始名称用于展示 |
| 自动播放被浏览器拦截 | 默认静音、`playsinline`、保留控件与播放按钮路径 |
| 旧 `代码与项目` 路由失效 | 添加重定向到 `deal-results-showcase` |
| 移动端灯箱溢出 | 灯箱移动端改为单列布局，视频高度限制在视口内 |
| 后续部署平台限制单文件大小 | 当前记录为风险；后续用压缩、CDN 或对象存储单独处理 |
