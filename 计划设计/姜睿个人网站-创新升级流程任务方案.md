# 姜睿个人网站 - 创新升级流程任务方案

## 目标

把当前个人简历网页从“模板化作品集”升级为一个有明确个人识别度的互动型技术页面。核心定位为：AI 设计工程师、FPV 飞手、全机型无人机飞手、程序员。页面表达不复制参考视频或 Active Theory，而是提炼其高级感来源：全屏沉浸主体、极简文字、滚动驱动变化、鼠标近场反馈、章节像空间节点一样被点亮。

## 参考拆解

### 视觉风格

1. 首屏应由一个强主视觉承担记忆点，而不是头像、卡片和按钮堆叠。
2. 参考视频用自然物体和材质生长制造差异化，本项目改为透明科技桨叶、航迹粒子、HUD 刻度、飞控数据感。
3. 页面背景保持深黑/石墨色，使用青绿色作为飞行路径和交互响应色，避免泛紫、米色、橙色和普通渐变。
4. 内容区域减少传统卡片网格，改为轨迹、节点、开放式信息带和证据库。

### 交互方式

1. 鼠标靠近主视觉时，粒子和光点产生近场扰动。
2. 页面滚动时，桨叶旋转速度、航迹线、导航胶囊和章节节点产生联动，形成“从飞行系统进入证据库”的叙事。
3. 首屏保留一个轻量滚动提示，指向下一个章节。
4. 移动端和微信内置浏览器降级为静态但完整的内容体验，避免 WebGL 卡顿或黑屏。

## 实施步骤

1. 资源归档与文本抽取
   - 将简历和参考视频复制到项目内稳定英文文件名。
   - 用 PDF 文本抽取和页面渲染确认简历内容。
   - 抽参考视频关键帧，分析视觉与交互，不直接搬运素材。

2. 内容模型重构
   - 修复现有中文乱码。
   - 根据简历提炼真实能力：CAAC 多旋翼教员、中型超视距多旋翼机长、中型超视距直升机机长、Python、嵌入式、智能体工作流、地面站、测试开发实习、竞赛经历。
   - 首页不直接暴露电话等敏感信息，详细联系方式保留在 PDF 和微信二维码中。

3. 视觉系统升级
   - 采用“AI Flight Lab”作为页面概念。
   - 主视觉改为透明科技桨叶 + 航迹粒子 + HUD 数据。
   - 章节从 About/Skills/Experience/Videos/Contact 变为 Mission、Systems、Timeline、Evidence、Contact 的连续技术叙事。

4. 交互升级
   - 保留 R3F 懒加载，避免首屏阻塞。
   - 增加滚动进度变量，用 CSS 和 Three.js 同时驱动视觉状态。
   - 增加章节节点 hover/active 感、证据库占位卡的“待补齐资源”状态。
   - 所有动效遵守 `prefers-reduced-motion`。

5. 资源坑位设计
   - AI 设计作品：预留图文/视频入口。
   - FPV 飞行视频：预留视频卡和未来本地视频路径。
   - 全机型无人机图库：预留机型/场景展示。
   - 代码项目：预留 GitHub 或项目链接。
   - 飞行日志/测试报告：预留 PDF/表格/截图入口。
   - 证书扩展：预留后续证书资源。

6. 验证
   - 运行 `npm run build`。
   - 使用 Playwright + 本机 Chrome 检查桌面和移动端渲染。
   - 检查首屏不空白、无框架报错、滚动和按钮交互有效、中文无乱码、资源正常加载。

## 风险预测与解决方案

| 风险 | 影响 | 解决方案 |
| --- | --- | --- |
| 过度复制参考视频 | 页面缺乏原创性，存在侵权和同质化风险 | 只借鉴交互逻辑，不使用树枝、苔藓、原文案、原排版；转译为无人机桨叶和航迹系统 |
| WebGL 性能过重 | 低端设备、微信内置浏览器可能卡顿或黑屏 | R3F 懒加载；移动端/微信/低 GPU 降级；粒子数量按设备分级 |
| 中文乱码 | 直接损害专业感 | 重写中英文本数据和 i18n 文件，统一 UTF-8 内容 |
| 简历敏感信息外露 | 电话、邮箱等信息被首页直接抓取 | 首页只保留微信二维码和 PDF 下载；详细信息由用户主动下载 PDF 查看 |
| 资源不足导致页面空 | 视频、项目图、AI 作品暂未补齐 | 用“待接入”证据库坑位明确说明资源类型，不伪造项目成果 |
| 简历能力表述过度包装 | 招聘方验证时出现不一致 | 所有能力来自简历或用户明确声明；不虚构飞行时长、项目数据、商业成绩 |
| 参考视频水印和平台内容 | 不能作为站点素材 | 仅用于分析交互，不嵌入页面、不公开展示为作品 |
| 后续资源持续更新 | 文件名和路径容易变乱 | 固定资源目录：`个人信息资源包/` 放原始资料，`public/` 放站点可访问资产，数据文件集中维护 |

## 本轮改造验收标准

- 中文内容无乱码。
- 首屏不再是普通模板，必须有透明科技桨叶和飞行轨迹主视觉。
- 页面包含 AI 设计工程师、FPV 飞手、全机型无人机飞手、程序员四个身份线索。
- 参考视频的交互方式被转译为滚动/鼠标/章节联动，而不是只换视觉风格。
- 简历 PDF 可以下载。
- 未来资源位置清晰，不伪造成品。
- 构建和浏览器渲染检查通过。
