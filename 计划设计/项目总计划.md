# 项目总计划

## 项目定位

项目名称：姜睿个人网站 / AI Flight Lab

目标：建设一个高级、创新、灵动的个人求职与作品展示网站，围绕以下身份建立可信叙事：

- AI 设计工程师
- FPV 飞手
- 全机型无人机飞手
- 程序员
- 无人机测试方向求职者

页面不做普通简历模板，而是做成可持续补充证据的互动型技术实验室：主视觉负责记忆点，内容负责真实能力，证据库负责后续资源接入。

   - 已抽帧到：`reference-video-screenshots/`

   - 已截取不少于 30 张图，保存于：`参考截图-activetheory/`
   - 用途：分析高级感、WebGL 沉浸感、暗色空间、少量文字、全屏主视觉。

   - 临时地址前期可用作参考。
   - 后续不再更新。

6. 当前页面方向：
   - 采用 `AI Flight Lab` 概念。
   - 首屏主视觉为透明科技感桨叶。
   - 交互包含鼠标近场粒子响应、两侧滚轮联动粒子、滚动动画、任务侧轨、能力节点切换、Mission Control 窗口切换和路由扫描转场。
   - 页面结构为：首页长滚动体验 + `Video Bay` 独立视频舱 + `Evidence Vault` 独立证据库。
   - 当前框架已迁移为 `Vite + Vue 3 + TypeScript + Vue Router`。
